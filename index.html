THE ANTIMICROBIAL PEPTIDE REPOSITORY

AMP-R is a repository of Antimicrobial Peptides (AMPs) consisting of a vast library of 5544 structural AMP models. This entire set of FASTA files, along with PDB structures and 3D peptide models, has been made available for use in this search-enabled repository. This also includes the 2547 filtered anti-Gram positive peptides, and results from protein-peptide docking studies. The database is searchable using DRAMP IDs or PEP IDs (please see files with IDs listed). AMP-R also contains in-house developed Python scripts used to filter peptides from the library of 5841 general AMPs, and these codes can be modified depending on use. Taken together, AMP-R is a user-friendly and accessible database of AMP structural models that can serve as a resource for future bioinformatics studies examining AMPs for antimicrobial and other biological applications.
