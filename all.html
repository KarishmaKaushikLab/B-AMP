AMP-R is an easy search facility for AMPs, using PEP ID, DRAMP ID, or name of the AMP, across a library of 5544 AMPs. Each card for a particular AMP contains its own FASTA file, PDB structure file and model image, which can be downloaded by clicking on the respective icons. Please click on the SHOW HELP button for specific instructions for search specifications and the image legend.
